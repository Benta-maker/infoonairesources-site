Security

Info on AI Resources welcomes responsible reports about vulnerabilities or security concerns affecting this website.

Report a Security Concern

Please email info@infoonairesources.shop with a clear description of the issue, affected URL, reproduction steps, and any relevant evidence.

Responsible Disclosure Guidelines

  • Do not access, modify, delete, or exfiltrate data that does not belong to you.
  • Do not perform denial-of-service testing, social engineering, spam, or physical attacks.
  • Give us a reasonable opportunity to investigate and address validated issues before public disclosure.
  • Act in good faith and avoid privacy violations or service disruption.

Security Controls

This static site is configured to use browser security headers, including a Content Security Policy, HTTPS enforcement through HSTS, clickjacking protection, MIME-sniffing protection, referrer limits, and restrictive browser permissions.

Machine-readable security contact information is available at /.well-known/security.txt.
